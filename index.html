üî¢ Data types

Python has several built-in data types that are fundamental to programming:

  • Integers (int): Whole numbers (e.g., 5, -10).
  • Floating-point (float): Decimal numbers (e.g., 3.14, 2.0).
  • Strings (str): Text sequences (e.g., "Hello").
  • Booleans (bool): True or False values.
